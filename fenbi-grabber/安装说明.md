# 粉笔错题数据抓取器 · v0.3 安装与使用

> 这一版是「纯一键」：在粉笔报告/解析页点一下按钮，直接下载 `错题集.docx`。
> 同时会下载一个 `fenbi-bundle-时间戳.json` 作为备份，可继续用旧命令行流程。
>
> 要求：Chrome 111 及以上（2023 年后的版本都行）。

## 一、安装（只需一次）

1. 地址栏输入 `chrome://extensions` 回车。
2. 右上角打开「开发者模式」。
3. 点「加载已解压的扩展程序」，选中**本文件夹**（`fenbi-grabber`，里面有 `manifest.json` 的那层）。
4. 列表里出现「粉笔错题数据抓取器」即装好。

## 二、生成错题集 Word

1. 在 Chrome 打开你做完的那套卷子的**报告 / 解析页**。
2. 看页面**右下角的蓝色按钮**，等它变成绿色「✅(2/2)」——说明 solution 和 getReport 都抓到了。
   - 正常情况下首次打开即可变绿，无需手动刷新。
   - 如果 10 秒后仍不是 (2/2)：再刷新一次；仍不行就按下面「排查」处理。
3. 点这个按钮：
   - 浏览器会下载 **`错题集.docx`**。
   - 同时会下载 **`fenbi-bundle-时间戳.json`** 作为备份（可忽略，留着防万一）。
   - 若弹「允许多文件下载」点允许。
4. 直接用 Word / WPS 打开 `错题集.docx` 即可。

## 三、回到旧流程（高级/调试）

如果 Word 有缺图或排版异常，可用备份 bundle 走旧命令行：

```bash
python3 unbundle.py fenbi-bundle-xxx.json
python3 parse.py solution.txt getReport.txt
node build_docx.js
```

## 四、排查

### 按钮一直不变绿
- 确认在的是**报告/解析页**（能看到题目对错和解析的那页），不是题目列表页。
- 确认刷新过；抓取器要在加载时就位才能拦到请求。
- 打开 Console 看有没有 `[FENBI-GRABBER]` 相关日志；把日志贴给我。

### Word 里有图片缺失
- 有些粉笔加密图 canvas 也拿不到，会在 Word 里显示橙色占位框，这是正常的。
- 如需补图，用备份 bundle 走「旧流程」出 Word，再按 `缺图清单.txt` 手动补。
