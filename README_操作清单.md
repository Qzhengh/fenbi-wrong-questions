# 粉笔错题集 · 操作清单（在你自己的 Mac 上跑）

这套工具把你导出的粉笔卷子数据，自动整理成一份排版好的 Word 错题集。
**你只需要做两件事：① 从粉笔导出两个文件 ② 跑两条命令。** 跟着下面做即可。

---

## 一、准备：导出两个文件（每套新卷子都要重新导）

在 Chrome 打开你做完的那套粉笔卷子的**报告/解析页**，然后：

1. 页面上右键 →「检查 / Inspect」→ 顶部点「Network」标签。
2. 按 **⌘ + R** 刷新，等条目刷出来，点一下「Fetch/XHR」过滤。
3. 导出 **solution**：
   - 在左边列表找到名字以 `solution?key=...` 开头的那一行，单击它；
   - 右边点「Response」→ 在内容上右键 →「Copy」→「Copy response」；
   - 打开「文本编辑」新建空白文档，粘贴(⌘V)，存成 **`solution.txt`**。
4. 导出 **getReport**：
   - 同样找到 `getReport?format=html...` 那一行，重复上面的复制动作；
   - 存成 **`getReport.txt`**。

> 提示：文本编辑如果是富文本也没关系，脚本能自动处理。
> 把 `solution.txt`、`getReport.txt` 和本工具的两个脚本放进**同一个文件夹**。

---

## 二、第一次使用：装好环境（只需一次）

打开「终端」(Terminal)，把下面命令逐行粘进去回车。

1. 检查是否已有 Node 和 Python（装了 Claude Code 一般都有）：
   ```
   node -v
   python3 -v
   ```
   两个都能显示版本号就行。若 `node` 没有，去 https://nodejs.org 装一个 LTS 版。

2. 安装两个脚本要用的库：
   ```
   npm install docx
   ```
   （Python 用的都是自带库，无需额外安装。）

---

## 三、每次生成错题集：跑两条命令

进入放文件的那个文件夹（把 `文件夹路径` 换成你的实际路径）：
```
cd 文件夹路径
```

**第 1 条** —— 解析数据 + 下载图片：
```
python3 parse.py solution.txt getReport.txt
```
跑完你会看到类似：
```
挑出错题 32 道。开始下载图片 ...
图片下载完成：成功 37 张，失败 0 张
✅ 已生成 errors.json
各板块错题数： {...}
```
- 如果「失败 0 张」→ 完美，所有图都下好了。
- 如果有「失败 N 张」→ 不要紧，这些图会在 Word 里显示成橙色虚线占位框，
  失败的网址会列在屏幕上（也存进了 errors.json 的 failed_list）。
  可以稍等几分钟网络稳定后**再跑一次**这条命令，通常就好了。

**第 2 条** —— 生成 Word：
```
node build_docx.js
```
跑完会得到 **`错题集.docx`**，用 Word 打开即可。

---

## 四、验收（重要，别省）

打开 `错题集.docx`，对照检查（这套机制是为了不让错误溜过去）：

1. **数量对账**：文档开头写了「共 N 题」。回粉笔报告页确认你做错的题数是不是这个数。
2. **图片/公式**：从头翻到尾，看有没有**橙色虚线占位框**（写着"图片缺失/公式图缺失"）。
   - 有 → 说明那张图没下成。重跑第1条命令再生成；若反复下不下来，记下题号，最后手动把图补进去。
3. **答案页**：文档最后是「参考答案与解析」。抽查两三道，看正文标的答案和解析里说的答案一致。
4. **资料分析**：看带【材料】【问题】的题，材料(含图表)在上、错的小题在下，对得上。

---

## 五、换新卷子怎么办

重复**第一步**导出新的 `solution.txt`、`getReport.txt`（覆盖旧的），
再跑**第三步**那两条命令即可。每套卷子生成的 `错题集.docx` 记得改个名字存好
（比如「错题集_江西卷.docx」），免得被下次覆盖。

---

## 附：判定规则（脚本怎么挑题，供你了解）

- 只收**做错**的题（粉笔数据里 status = -1）。
- **不收**你做对的（status = 1）。
- **不收**你没作答/整段跳过的（status = 10）。
- 答案字母换算：粉笔用 0/1/2/3 表示 A/B/C/D，脚本已自动转好。

有任何一步卡住，把终端里的报错信息复制下来，连同这份说明发给 Claude 即可。
